# ChromaSketch | Premium Coloring Book App

ChromaSketch is a modern, high-performance web application that transforms any image into a fillable coloring book page. 

[View Project on GitHub](https://github.com/ibrezm1/colorbook) | [**Live Demo**](https://ibrezm1.github.io/colorbook/)

## 🚀 Features

- **Smart Image Processing**: Automatically converts photos to black & white line art.
- **Layered Painting**: Color underneath the lines so your artwork stays clean.
- **Flood Fill & Pen**: Use the bucket for large areas and the pen for fine details.
- **Recent Colors**: Keeps track of your last 5 used colors for quick access.
- **Custom Themes**: Switch between Dark and Light mode.
- **Collapsible Sidebar**: Maximize your workspace with a single click.
- **Export Art**: Save your masterpieces as high-quality PNGs.

## 🛠️ Built With

- **HTML5 Canvas**: For high-performance image manipulation and drawing.
- **Vanilla JavaScript**: Pure, fast logic with no heavy frameworks.
- **Modern CSS**: Custom properties, glassmorphism, and responsive design.
- **Lucide Icons**: Clean, consistent iconography.

## 📦 Deployment
